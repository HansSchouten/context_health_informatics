Compares the data of two columns for time difference or value difference.

Syntax

  • COMPARE MEASURMENTS COL(actual) COL(entered)
  • COMPARE COL(value1) COL(value2)
  • COMPARE PATTERN [pattern]

Examples

  • COMPARE COL(DateTime) COL(otherDateTime)
    Compares the time stamps of the date columns and returns the difference in time.
  • COMPARE PATTERN LABEL(X) WITHIN(2) LABEL(Y) AFTER(3) LABEL(Z)
    Counts the number of occurrences of the pattern: label x, within 2 days label y, 3 days after label y the label z.
