Adds labels to created data records, data elements or data chunks to mark them with keywords.

Syntax

  • LABEL(label)
  • LABEL WITH (label) WHERE COL(column) condition
  • LABEL WITH labelname WHERE [condition] AFTER PATTERN [pattern]

Examples

  • LABEL WITH filled WHERE COL(Level) > 0
    Labels every record with a level higher than zero with 'filled'.
  • LABEL WITH measured_right WHERE COL(phase) = 1 AFTER PATTERN LABEL(entered) AFTER(1) WITHIN(2) LABEL(entered)
    Labels every record that is entered one day after the last entered record with 'measured_right', but only if the pattern ends with a record that satisfies the condition: phase equals 1.
