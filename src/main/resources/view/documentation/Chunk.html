Chunking is used to group a large set of data into a set of smaller chunks.

Syntax

  • CHUNK ON COL(column)
  • CHUNK ON COL(column) PER number DAYS
  • CHUNK PER number DAYS
  • CHUNK PATTERN [pattern]
    Chunking on a pattern results in a chunk for each occurrence of a pattern. The chunk starts with the record containing the first label of the pattern and will continue untill the record containing the last label of the pattern.
  • CHUNK FLATTEN
    Combines all records of each chunk into one record per chunk.
  • CHUNK REMOVE
    Closes the chunking and transforms the data back to a sequential data structure.

Examples

  • CHUNK ON COL(Level) PER 7 DAYS
    Chunks the data of the Level column for every 7 days since the oldest date.
  • CHUNK PATTERN LABEL(X) WITHIN(2) LABEL(Y)
    Divides the data in chunks starting from label x and within 2 days stops at label y.
