Variables are used to store a temporary result for later use in the script.

Syntax

  • $name = operation
  • operation USING $name

Examples

  • $VAR = FILTER WHERE COL(level) > 100
    Stores the rows where the level column is higher than 100 in the VAR variable.
  • FILTER WHERE COL(level) < 150 USING $VAR
    Filters the rows where the level column is lower than 150 using the data from the VAR variable.
