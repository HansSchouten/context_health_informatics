Computation is used to perform computations on the data:
AVG, COUNT, SUM, MAX, MIN, DEVIATION, SQUARED, VARIANCE

Syntax

  • COMPUTE computation COL(value)

Examples

  • COMPUTE AVG COL(level)
    Computes the average level value of the selected records.
  • COMPUTE MAX COL(level)
    Computes the maximum level value of the selected records.
