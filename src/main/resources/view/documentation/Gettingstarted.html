Getting started

Introduction

This application was created in a way to guide you into a natural simple workflow for analyzing large amounts of sequential data. You have to follow some steps before you can create the result you want, which will be described below.

Import

Before starting the analysis, you obviously need to import some data. We do this by creating groups of files in the same format. For every type of file you should create a new group, even if it’s just a single file. Files can be added by either pressing the Browse button or by dragging files into the Files section.

Now it’s time to specify the columns. If you picked the right delimiter next to the group name, you should see an entry field for each column in the Columns section. Here you can select the type and add a name. We expect at least one column to specify a date or time.

You also have to specify a Primary Key for each group. This is used to identify to whom or what the data belongs, for example the user ID of a person. You can select either a column or the file name to as primary key. When the data of a group doesn’t belong to anything in particular, you can select the No Primary Key option.

Select

In the select tab you can select the primary keys of the data you want to analyse. Pretty straightforward!

Analyse

For analyzing the selected data, we created a scripting language to easily and effectively perform operations. In the Analyse tab itself you can see some documentation of the operations in the bottom left. The operations are executed in a pipeline-like fashion: The output of one operation is in the input of the next.

You can also create variables, these are using the input of the previous operation but do not affect the next operation. These variables are listed in the sidebar, and can be opened by double clicking.

Result

When you have finished your modifications on the data, you can view the result in this tab. You are presented with a table of the result, which can be rearranged by dragging the column headers and sorted by clicking on the column headers. You can hold shift and click to sort on multiple columns.

If you want to modify the table as text, press the Convert To Text button. In this section you can modify the text to your liking and save it to a file.

In the last section you can create graphs. These can visualize the data to easily find patterns or other information.
