Convert is used to automatically generate behavior of web site response.

Syntax

  • CONVERT PHASE
  • CONVERT COL(column)
  • CONVERT REMEASUREMENT COL(column)
  • CONVERT SECOND MEASUREMENT COL(column)

Examples

  • CONVERT PHASE
    Adds a column 'phase' that indicates in which phase the patient currently is in.
  • CONVERT COL(creatinine) PER 7 DAYS
    Adds the columns 'grensgebied', 'kreatinine status' and 'feedback' containing the status of a patient based on the creatinine values of the last two days.
  • CONVERT REMEASUREMENT COL(creatinine) PER 7 DAYS
    Adds a column 'remeasurement' and set it to true if this record represents a remeasurement as advised on the previous day.
  • CONVERT SECOND MEASUREMENT COL(creatinine)
    Adds a column 'second measurement' and set it to true if this record represents a remeasurement as advised earlier this day.
