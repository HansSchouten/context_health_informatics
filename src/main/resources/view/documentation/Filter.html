Filtering is used to constrain the data based on a condition.

Syntax

  • FILTER WHERE (condition)

Examples

  • FILTER WHERE (COL(level) > 150) AND (COL(status) > 2)
    Filters the data on a creatinine level higher than 150 and daily status higher than 2 and returns the corresponding records.
  • FILTER WHERE LABELED(X)
    Selects only the records that have the label X.
